# Amazon_clone
Developed a responsive Amazon-inspired e-commerce interface using HTML, CSS, and JavaScript, focusing on clean UI design, structured layouts, and interactive components to simulate a real-world shopping experience.
## Images
Image assets are intentionally excluded from this repository  to keep it lightweight.The project structure and references are maintained,allowing images to be added if required.
